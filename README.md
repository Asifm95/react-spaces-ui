## Installation

    $ git clone https://github.com/Asifm95/react-spaces-ui.git
    $ cd react-spaces-ui
    $ npm install

## Start & watch

    $ npm start

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Simple build for production

    $ npm run build
